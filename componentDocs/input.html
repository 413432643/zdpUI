# input 输入框

通过鼠标或键盘输入字符

# 基本用法

点击查看代码
<z-input v-model="input" placeholder="基本用法"></z-input>


<script setup>
import { ref } from "vue";
const input = ref('')
</script>

# 禁用状态

通过 disabled 属性指定是否禁用组件

点击查看代码
<z-input placeholder="禁止状态" disabled></z-input>

# 一键清空

使用clearable属性即可键清空输入框

点击查看代码
<z-input placeholder="一键清空" clearable></z-input>

# 密码框

设置 type属性为password 即可得到一个可切换显示隐藏的密码框

点击查看代码
<z-input placeholder="密码框" clearable></z-input>

# 带图标的输入框

设置leftIcon属性为图标名称即可得到左侧带图标的输入框 设置rightIcon属性为图标名称即可得到左侧带图标的输入框

# 文本域

设置 type属性为textarea 即可得到一个用于输入多行文本信息可缩放的输入框

点击查看代码
   <z-input v-model="textarea" placeholder="文本域" type='textarea' rows="2" ></z-input>

# 自适应高度文本域

添加autosize属性即可让输入框自动调整高度

点击查看代码
  <z-input v-model="textarea" placeholder="文本域" type='textarea' rows="1" autosize></z-input>

# input 属性

属性说明类型可选值默认值
v-model输入框绑定值string/number--
disabled是否禁用boolean--
placeholder占位文本string--
clearable是否显示清除按钮boolean--
type输入框类型stringtext/password/textareatext
leftIcon左侧图标string--
rightIcon右侧图标string--
rows文本域行数(只在type='textarea'生效)number--
autosize文本域是否自适应高度(只在type='textarea'生效)boolean--
interval输入框间隔时长(单位ms)number-500

# Input 事件

事件名说明参数
input输入框输入事件
focus输入框获得焦点事件-
blur输入框失去焦点事件-
clear清除按钮点击事件-
change仅在输入框失去焦点或用户按下回车时触发-
