# 快速开始


### 安装组件库

```bash
npm i zdp-ui
```

### 引用组件库

```javascript
// 全部引用
import 'zdp-ui/dist/assets/index.73364d44.css'
import ZUI from 'zdp-ui'
Vue.use(ZUI)

// 按需引用
import 'zdp-ui/dist/assets/button.css'
import {Button} from 'zdp-ui'
Vue.use(Button)

```
